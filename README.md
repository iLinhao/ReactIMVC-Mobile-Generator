# WealthMobile-Generator

ReactIMVC Mobile程序骨架生成器，基于Yeoman脚手架工具规范ReactIMVC Mobile程序开发并提高开发效率。

## 使用包管理工具安装yo
使用npm：
```
npm install -g yo
```
使用yarn：
```
yarn global add yo
```
## 链接到npm全局环境
```
cd ReactIMVC-Mobile-Generator

npm install

npm link
```
## 新建Feature项目目录并使用generator初始化代码
```
cd feature-xxx
yo reactimvcmobile
```
